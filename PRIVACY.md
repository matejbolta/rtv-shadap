# Privacy Policy

RTV Shadap does not collect, transmit, sell, or share user data.

The extension runs only on `https://www.rtvslo.si/`. It uses local browser storage to remember article IDs, article status, and timestamps so it can show whether RTV SLO homepage stories are new, already seen, opened, or live.

This data stays on the user's device in `chrome.storage.local`. RTV Shadap has no server, analytics, ads, telemetry, or remote code.

Users can delete the extension's local history from the extension popup by selecting **Ponastavi zgodovino**. Users can also remove all extension data by uninstalling the extension from Chrome or Brave.

RTV Shadap is an independent browser extension and is not affiliated with RTV SLO.
